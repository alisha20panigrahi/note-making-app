# note-taking-nodejs



------------------------------------------------------------------
| How to use? -                                                  | 
|                                                                |
| ADD Note:                                                      |
|    node note.js add --title="TITLE" --body="BODY"              |
|                                                                |
| REMOVE Note:                                                   |
|    node note.js remove --title="TITLE"                         |
|                                                                |
| LIST Notes:                                                    |
|    node note.js list                                           |
|                                                                |
| READ Notes:                                                    |
|   node note.js read --title="TITLE"                            |
|                                                                |
------------------------------------------------------------------
